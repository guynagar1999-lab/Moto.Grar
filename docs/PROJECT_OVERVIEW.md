# 📋 סקירת פרויקט - MotoGrar

**MotoGrar** הוא אתר אינטרנט מתקדם וחדשני שנבנה עבור עסק גרירת אופנועים, ATV ו-RZR/UTV. האתר משלב טכנולוגיה מתקדמת עם עיצוב מרשים וחוויית משתמש יוצאת דופן, במטרה להפוך את העסק לדיגיטלי ולמשוך לקוחות חדשים.

### 👤 בעל העסק
- **שם:** גיא נגר
- **תחום:** גרירת אופנועים, ATV ו-RZR/UTV
- **אזור שירות:** כל רחבי הארץ
- **זמינות:** 24/7
- **טלפון:** 052-482-3435

## 🌟 מה האתר מכיל?

### 📄 דפים ראשיים

#### 1. **דף הבית** (`/`)
- **Hero Section** מרשים עם אנימציות מתקדמות
- הצגת השירותים העיקריים (אופנועים, ATV, RZR)
- טופס בקשת שירות מהיר
- פיצ'רים מרכזיים (זמינות 24/7, ציוד מתמחה, ביטוח מלא)
- קישורים מהירים לכל השירותים

#### 2. **עמודי שירותים** (`/services/*`)
- **אופנועים** (`/services/motorcycle`) - גרירה מקצועית לאופנועים
- **ATV** (`/services/atv`) - חילוץ וגרירה לכלי ATV
- **RZR/UTV** (`/services/rzr`) - שירותים לכלי RZR ו-UTV כבדים

כל עמוד שירות כולל:
- תיאור מפורט של השירות
- מחירים משוערים
- רשימת היתרונות
- טופס הזמנה עם אפשרות התקשרות/וואטסאפ
- שירותים נוספים בתחום

#### 3. **עמוד ביקורות** (`/testimonials`)
- ביקורות לקוחות אמיתיות עם תמונות פרופיל
- דירוג כוכבים (5/5 בממוצע)
- פירוט השירותים שקיבלו
- תאריכי הביקורות
- קריאה לפעולה להצטרפות ללקוחות מרוצים

#### 4. **עמוד צור קשר** (`/contact`)
- טופס צור קשר מתקדם עם ולידציה
- פרטי התקשרות מלאים (טלפון, וואטסאפ, אימייל)
- מידע על אזור השירות והזמינות
- קישורים ישירים להתקשרות

#### 5. **אזור Pro Garage** (`/pro-garage`)
- תוכן מתקדם למכונאים מקצועיים
- מדריכי תיקון ותחזוקה מפורטים
- גייטד קונטנט (גישה מוגבלת)
- רמות קושי (מתחיל, בינוני, מתקדם)
- חלקים נדרשים וזמני ביצוע

### 🎮 תכונות אינטראקטיביות

#### 🤖 **Chatbot AI** (`/chatbot`)
- עוזר וירטואלי חכם עם RAG (Retrieval Augmented Generation)

- מבוסס על OpenAI GPT API

#### 🎯 **משחק דפדפן** (`/game`)
